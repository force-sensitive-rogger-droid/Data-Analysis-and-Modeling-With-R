## 📌 Executive Summary
This repository showcases advanced data science workflows in **R**, moving beyond simple analysis into **Probabilistic Modeling** and **Machine Learning**. 

The portfolio features high-volume data wrangling (50k+ records), Bayesian parameter estimation using **Stan**, and predictive modeling with **Random Forests**. Each project focuses on converting raw urban and market data into actionable business intelligence.

---

## 📂 Project Portfolio

### 1. [Bayesian Linear Regression & Predictive Inference using Stan](https://github.com/force-sensitive-rogger-droid/Data-Analysis-and-Modeling-With-R/tree/main/Bayesian%20Linear%20Regression%20%26%20Predictive%20Inference%20using%20Stan)
* **Core Tech:** `rstan`, `brms`, `bayesplot`, `MCMC`
* **Analysis:** Implemented Hamiltonian Monte Carlo (NUTS) to model household expenditure. 
* **Key Skills:** Convergence diagnostics (R-hat, ESS), Prior sensitivity analysis, and Posterior Predictive Checks (PPC).

### 2. [Multi-Source Data Analysis & Predictive Modeling with R](https://github.com/force-sensitive-rogger-droid/Data-Analysis-and-Modeling-With-R/tree/main/Multi-Source%20Data%20Analysis%20%26%20Predictive%20Modeling%20with%20R)
* **Core Tech:** `caret`, `randomForest`, `dplyr`, `tidyr`
* **Analysis:** Integration of SCATS traffic volumes and Twitch streamer metrics.
* **Key Skills:** Feature engineering, hyperparameter tuning (`ntree` optimization), and classification performance evaluation (Confusion Matrix, Kappa).

### 3. [Urban Mobility & Traffic Flow Analysis](https://github.com/force-sensitive-rogger-droid/Data-Analysis-and-Modeling-With-R/tree/main/Urban%20Mobility%20%26%20Traffic%20Flow%20Analysis)
* **Core Tech:** `lubridate`, `ggplot2`, `scales`, `gridExtra`
* **Analysis:** Large-scale joining of DublinBikes occupancy and SCATS sensor data.
* **Key Skills:** Handling time-series data, identifying urban bottlenecks, and multi-dataset synchronization.

### 4. [Probabilistic Comparative Analysis: A Bayesian Approach](https://github.com/force-sensitive-rogger-droid/Data-Analysis-and-Modeling-With-R/tree/main/Probabilistic%20Comparative%20Analysis%3A%20A%20Bayesian%20Approach)
* **Core Tech:** `extraDistr`, `stats`, `Base R`
* **Analysis:** Comparative study using Beta-Binomial conjugate frameworks.
* **Key Skills:** Bayesian hypothesis testing, Posterior Odds calculation, and Credible Interval estimation.

* **Probabilistic Programming:** **Stan** (via `rstan`) for high-performance statistical computing.
